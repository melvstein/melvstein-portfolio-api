# Melvstein Portfolio API

This is my personal backend project built with **Spring Boot**.  
It serves as my portfolio API where I practice and demonstrate backend development skills, system design, and modern Java ecosystem tools.

This project is not intended for public or production use — it is mainly for learning, experimentation, and showcasing my development style.

---

## 🚀 Tech Stack

### Core Framework
- Spring Boot (Web, Security, Validation)
- Spring Web (REST API development)
- Spring Security (authentication & authorization)

### Data & Persistence
- Spring Data JPA (Hibernate ORM)
- PostgreSQL Driver
- Flyway (database migrations)

### Performance & Messaging
- Spring Data Redis (caching layer)
- Spring for Apache Kafka (event-driven messaging)

### Developer Experience
- Lombok (reduce boilerplate code)
- Spring Boot DevTools (hot reload)
- Docker Compose Support (local environment setup)

### API Documentation
- SpringDoc OpenAPI (Swagger UI)

---

## 📌 Purpose of This Project

I built this project to:

- Practice building scalable Spring Boot applications
- Improve my understanding of backend architecture
- Work with real-world tools like Redis, Kafka, and PostgreSQL
- Learn proper API design and documentation
- Experiment with authentication and security setups
- Strengthen my knowledge in database migrations using Flyway
